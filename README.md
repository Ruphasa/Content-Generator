# Venturo Pro AI Content Generator

An interactive, Next.js-based web application that streamlines the process of generating AI content (Videos, Websites, Campaigns). 

## Features
- **3-Column Brand Card Layout:** A modern, non-linear form for inputting comprehensive brand identity, visual style, tone of voice, and business details.
- **Sidecar AI Copilot:** A context-aware assistant that guides you in real-time as you fill out the DNA form.
- **Glassmorphism UI:** Premium styling featuring Venturo's signature Teal (`#009BAD`) with sleek frosted-glass effects.
- **Seamless Asset Uploads:** Integrated drag-and-drop zones directly within the conversational flow.

## Tech Stack
- [Next.js](https://nextjs.org/) (App Router)
- [React](https://react.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/)
- [Vercel AI SDK](https://sdk.vercel.ai/) & [Groq](https://groq.com/) (LLM Chatbot)
- [Supabase](https://supabase.com/) (Auth & Storage)
- [Bun](https://bun.sh/)

## Getting Started

First, install dependencies:

```bash
bun install
```

Run the development server:

```bash
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
